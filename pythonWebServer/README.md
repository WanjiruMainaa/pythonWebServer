# python Web Server Project
A collaborative project that involves creating a simple HTTP server in python, based on understanding web sever fundamentals.

## Project Overview
This project demonstrates core web server concepts including:
- Handling HTTP Get requests
- Serving static files
- Directory listing
- Dynamic content generation,
- CGI script execution


### Prerequisites
- Python3 installed

### Running Locally 
```bash
# clone repository
git clone https://github.com/YOUR-USERNAME/pythonWebServer.git

# (Optional but recommended) Create a virtual environment
python -m venv env
.\env\Scripts\activate on windows
source env\bin\activate on linux/mac
```

```
### Making Contributions

3. Make your changes
4. Test locally
5. Create a Pull Request
6. Request review from project lead
7. Merge after approval
```
